## CPU란?
### CPU
- Central Processing Unit.<br/>
- 주요 연산을 담당하는 컴퓨터의 제어 장치.<br/>
- [인터럽트] [1]를 통해 여러 작업을 동시에 진행할 수 있다.<br/>
- [가상 메모리]를 사용한다.<br/>

### 구성 요소
- 산술논리연산장치(ALU): 산술, 논리 연산을 하는 회로 장치.<br/>
- 제어 장치(Control Unit, CU): 명령어를 해석하며 데이터 처리 순서를 결정하고 프로세스를 조작한다.<br/>
- 레지스터: CPU 안에 있는 매우 빠른 임시 기억 장치.<br/>
