## 네트워크에는 어떤 유형이 있나요?
### 네트워크 유형
1. LAN<br/>
    - Local Area Network.<br/>
    - 여러개의 MAC 주소와 하나의 IP로 구성되는 네트워크.<br/>
    - 안정성이 높고 속도가 빠르다.<br/>
    <br/>
   
2. MAN<br/>
    - Metropolitan Area Network. 도시와 도시의 통신망.<br/>
    - LAN이 ``라우터``, ``브리지`` 등으로 연결된 형태.<br/>
    <br/>
   
3. WAN<br/>
    - Wide Area Network.<br/>
    - 인터넷이라고도 한다.<br/>
    <br/>
