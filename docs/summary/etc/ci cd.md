## CI/CD란?
### CI
- Continuous Integration.<br/>
- 개발자가 코드를 자주 통합할 수 있게 한다.<br/>
- 자동적인 빌드, 테스트, 병합 과정을 거친다.<br/>

### CD
- Continuous Delivery/ Deploy.<br/>
- 배포를 수동/ 자동으로 진행한다.<br/>

### CI/CD의 이점
- 개발 속도 향상.<br/>
- 버그를 빠르게 발견하고 해결.<br/>
- 코드 품질 관리.<br/>
