## Var vs Val vs Const
### Var
- Variable.<br/>
- **가변 변수**를 선언할 때 사용.<br/>

### Val
- Value.<br/>
- **불변 변수**를 선언할 때 사용.<br/>
- Java의 final과 유사.<br/>

### Const
- Constant.<br/>
- **상수**를 선언할 때 사용.<br/>
- 컴파일 타임에 값이 정해진다.<br/>
- 원시 타입과 String 타입에만 사용할 수 있다.<br/>
- Global 영역과 Static 영역(Companion Object 내부)에서, 또는 Object 선언 시에만 사용할 수 있다.<br/>
