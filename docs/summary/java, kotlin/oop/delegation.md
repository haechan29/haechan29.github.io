## 위임이란?
### 위임
- [구성] [1]의 한 방법으로, 객체가 특정 작업을 처리하기 위해 다른 객체의 메서드를 호출하는 것.<br/>
- 상속하지 않으면서 상속하는 것처럼 코드를 재사용할 수 있다.<br/>
- 위임 객체를 호출하는 보일러 플레이트가 발생한다. (Kotlin에서는 [by 키워드] [2]를 통해 보일러 플레이트를 없앨 수 있다.)<br/><br/>
